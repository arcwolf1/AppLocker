# AppLocker - 开源 Windows 应用锁（触屏友好，独立密码）

AppLocker 是一款轻量级、开源的 Windows 应用程序锁定工具，专为教学、演示、家长控制等场景设计。通过在程序启动前弹出密码验证窗口，防止未经授权的使用，并在程序退出后自动恢复锁定状态。特别针对触摸屏设备进行了优化，适合在 Windows 平板或电子白板上运行。

## ✨ 核心特性

- **独立密码** 每个被保护的程序可设置不同密码，互不干扰。
- **伪装无痕** 将原程序原地替换为启动器，**复制原始图标和文件名**，学生无法从外观识破。
- **无延迟** 采用轻量级拦截机制，不会像传统应用锁那样导致程序启动卡顿。
- **触摸友好** 密码框按钮大，UI 清晰，自动唤起屏幕键盘，支持回车确认输入。
- **密码隐藏** 输入时密码自动显示为黑点，防止旁人窥视。
- **命令行透传** 完美支持带参数启动的程序（如 `brave.exe --incognito`），参数原样转发。
- **自动恢复** 被锁定的程序退出后，启动器会自动交换文件，恢复下次启动仍需密码的状态。
- **修改与还原** 支持随时修改密码，或验证密码后还原为原始程序，不留痕迹。
- **详细日志** 所有关键操作均记录到 `applocker.log`，方便排查问题。
- **完全开源** 代码透明，无后门，放心使用。

## 🖥 运行环境

- **操作系统**：Windows 7 SP1 及以上（推荐 Windows 10/11）
- **.NET 运行时**：无需额外安装（内置 .NET Framework 4.7.2 支持）
- **权限要求**：伪装/还原程序时**需要管理员权限**（修改文件及替换图标），日常解锁无需管理员。

## ⚙ 快速上手

1. 从 [Releases](https://github.com/arcwolf1/AppLocker/releases) 下载最新的 `AppLocker.exe`（单文件，已包含所有依赖）。
2. **右键 → “以管理员身份运行”** 打开管理工具。
3. 点击 **「伪装程序」**，选择一个要锁定的 `.exe` 文件（如 `教学软件.exe`）。
4. 为该程序设置一个**独立密码**（两次输入确认）。
5. 勾选确认后，工具会自动：
   - 将原程序重命名为 `教学软件.exe.bak`（备份）
   - 将自身复制到该位置并改名，替换为原程序的图标
6. **即刻生效**：直接双击原程序（或原来指向它的任何快捷方式），会弹出密码框。
7. 输入正确密码后，原程序正常启动；退出后保护自动恢复。

其他功能：
- **还原程序**：在管理工具列表选中已伪装程序，点击「还原程序」，验证密码后恢复原始文件。
- **修改密码**：选中条目，点击「修改密码」，先验证旧密码，再设置新密码。
- **创建快捷方式**（备用）：对于不想原地替换的程序，可以生成桌面快捷方式指向启动器。

## 🧑‍🏫 适用场景

- **学校机房 / 多媒体教室**：仅老师知晓密码，学生无法随意打开聊天、游戏等无关程序。
- **触摸屏设备**：公共查询机、电子白板，防止误操作或退出到桌面。
- **家长控制**：锁定特定应用，限制儿童使用时间。

## 🛠 技术实现

- 开发语言：C#（Windows Forms）
- 目标框架：.NET Framework 4.7.2（Win10/11 自带，开箱即用）
- 单文件打包：[Costura.Fody](https://github.com/Fody/Costura) 将依赖 DLL 嵌入 EXE
- 密码哈希：SHA256（密码存储不包含明文）
- 图标替换：Windows API `UpdateResource` 动态修改可执行文件图标
- 文件恢复：原位重命名 + 批处理脚本监测进程退出

## 🔨 如何编译

```bash
# 克隆仓库
git clone https://github.com/your-repo/AppLocker.git

# 使用 Rider 或 Visual Studio 打开 AppLocker.sln
# 添加 COM 引用：Windows Script Host Object Model（用于创建快捷方式，如不需要可删除相关代码）

# 还原 NuGet 包（Costura.Fody）
nuget restore

# 生成 Release 版本
```

生成的 `AppLocker.exe` 位于 `bin/Release` 目录，可直接复制到目标电脑。
